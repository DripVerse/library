If you don't know what IPFS is, please learn more about [IPFS](/guide/nft/storage/ipfs) before continuing.

## Upload File to IPFS
```sh
drip upload <file>
```

## Upload All Files in a directory to IPFS
```sh
drip upload -r <dirpath>
```
