## Initialise
```sh
drip init
```

### Update values on config file on your favorite editor:
```sh
vim ~/.drip
```

By default, `SCOPE` is `'alpha'`. If you want to use mainnet, please change this to `'mainnet'`. As well as the `ALCHEMY_NETWORK_KEY` from `'maticmum'` to relevant key.
