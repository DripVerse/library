## Import

```js
import { Web3Storage } from 'dripverse';
```

or

```js
const { Web3Storage } = require('dripverse');
```

## Initialise

```js
const web3storage = new Web3Storage({token: process.env.NFT_STORAGE_API_KEY});
```
